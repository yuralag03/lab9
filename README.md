## 1. Алгоритм и блок схема:
### Алгоритм:
1. **Начало**
2. Объявить переменные:
 	- `a` — промежуточное вычисление
	- `b` — промежуточное вычисление
	- `c` — промежуточное вычисление
  	- `d` — промежуточное вычисление
   	- `e` — итоговое вычисление
  	- `x` — входное число
    - `y` — -1.625 * pow(10, 3)
   	- `z` — 0.512
3. Ввод переменных:
   - Ввод переменной: `x`
4. Вычислить:
   - `a` = exp(`x` - 1 / sin(`z`))
   - `b` = pow(`a`, 1.0 / 3.0)
   - `c` = sqrt(`x` + pow(fabs(`y`), 1.0 / 4.0))
   - `d` = pow(2, -`x`)
   - `e` = `b` * `c` * `d`
5. Вывести результаты расчётов с подстановкой значений в текст.
6. **Конец**

## 2. Реализация программы:
	#include <locale.h>
	#include <stdio.h>
	#include <math.h>

	int main() {
    setlocale(LC_ALL, "");

    double a, b, c, d, e, x, y;

    double z = 0.512;
    y = -1.625 * pow(10, 3);

    printf("Введите x: ");
    scanf("%lf", &x);           

    a = exp(x - 1 / sin(z));
    b = pow(a, 1.0 / 3.0);
    c = sqrt(x + pow(fabs(y), 1.0 / 4.0));
    d = pow(2, -x);
    e = b * c * d;

    printf("Ответ: %.5f", e);

    return 0;
	}
