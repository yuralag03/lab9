## Условия задачи:
Написать программу, выводящую на экран заданную геометрическую фигуру, нарисованную с помощью заданного с клавиатуры символа. Предусмотреть возможность изменения размеров фигуры путем задания ее размеров в количестве символов пользователем: ![Фигура](usl.png)
## 1. Алгоритм и блок схема:
### Алгоритм:
1. **Начало**
2. Объявить переменные:
 	- `symbol` — символ из которого состоит фигура
	- `r` — радиус
	- `continue_program` — флаг продолжение программы
  	- `size` — промежуточное вычисление
3. Ввод переменных:
   - Ввод переменной: `symbol`
   - Ввод переменной: `r`
4. Вычислить:
   - `size` = 2 * `r`
5. Цикл for `i` = 0 до `size`:
		Цикл for `j` = 0 до `size`:
			`x` = `j` - `r` - координата относительно центра
   			`y` = `i` - `r` - координата относительно центра
   			`dist_sq` = `x` * `x` + `y` * `y` - квадрат расстояния до центра
   			Проверка:
   					`on_square` = (`i` == 0 OR `i` == `size` OR `j` == 0 OR `j` == `size`)
   					`on_circle` = (`dist_sq` >= `r` * `r` - `r` AND `dist_sq` <= `r` * `r` + `r`)
   		Если `on_square` OR `on_circle`:
   			Вывести `symbol`
   		Инчае:
   			Вывести пробелы

6. Проверка продолжение:
   Ввод `continue_program`
   Если `continue_program` != 0:
   Запустить заново
   Иначе:
   		Вывести: "Программа завершена" 
7. **Конец**

## 2. Реализация программы:
```
#define _CRT_SECURE_NO_DEPRECATE
#include <locale.h>
#include <stdio.h>
#include <stdlib.h>

void main()
{
    setlocale(LC_ALL, "");
    int continue_program;

    do {
        char symbol;
        int r;
        int i, j;
        int size;

        printf("Введите символ из которого будет строиться фигура: ");
        scanf(" %c", &symbol);

        printf("Введите радиус r: ");
        scanf("%d", &r);

        size = 2 * r;

        printf("Квадрат с вписанной окружностью (r=%d):\n", r);

        for (i = 0; i <= size; i++)
        {
            for (j = 0; j <= size; j++)
            {
                int x = j - r;
                int y = i - r;

                int dist_sq = x * x + y * y;

                int on_square = (i == 0 || i == size || j == 0 || j == size);

                int on_circle = (dist_sq >= r * r - r && dist_sq <= r * r + r);

                if (on_square || on_circle) {
                    printf("%c", symbol);
                }
                else {
                    printf(" ");
                }
            }
            printf("\n");
        }

        printf("\nПродолжить? (Да - введите любое число. Нет - введите 0): ");
        scanf("%d", &continue_program);
        printf("\n");

    } while (continue_program != 0);

    printf("Программа завершена.\n");
    system("pause");
}
```
